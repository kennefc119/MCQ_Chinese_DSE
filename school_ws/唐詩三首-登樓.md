| **學習重點：**  1 內容 (1) 藉登樓遠望景色，抒發憂國憂民的情懷。  (2) 表達對朝廷的信心，以及自己報國無門的遺憾。  2 作法 (1) 襯托：以繁花美景反襯詩人悽愴的心境。  (2) 象徵：以「浮雲」象徵世事的變遷及政權的興亡，以「日暮」象徵年華老去。  (3) 用典：善用典故，含蓄表達詩人的心意。 |

**一、作者簡介**

- 杜甫（712–770），字子美，原籍襄陽（今湖北 襄樊），唐代詩人。曾在長安近郊的少陵居住，故自號少陵野老，人稱杜少陵。
- 生平：年青時漫遊各地，曾參加科舉考試落第。安 史之亂時被俘，逃出後任肅宗左拾遺，不久被貶為華州司功參軍。後來棄官入蜀，投靠劍南節度使嚴武，曾任檢校工部員外郎，世稱杜工部。嚴武去世後，杜甫離開成都，輾轉遷居夔州（今四川 奉節）。晚年飄泊無依，病死於湘江（位於今湖南）。
- 風格：杜甫作詩兼擅眾體，尤長於律詩和五、七言古體。他的詩歌內容寫實，多反映唐代由盛轉衰的變化，後人譽為「詩史」。由於他的詩充滿悲天憫人、憂國憂民的情懷，又被譽為「詩聖」。杜甫詩最主要的風格是沉鬱頓挫，作品筆法曲折多變。

**二、寫作背景**

- 《登樓》是杜甫在唐代宗 廣德二年（764）春寫於成都的，當時杜甫流落成都已經五年。前一年安史之亂雖然大致平定，但不久外族吐蕃攻陷長安，代宗被迫遠走。幸得郭子儀恢復京師，迎代宗復位，但吐蕃隨即又入侵今四川北部各地。當時唐朝外有吐蕃不斷侵擾，內則代宗昏庸，寵信宦官程元振、魚朝恩等，朝政日趨腐敗。國家內外交困，呈日益衰敗的景象。杜甫借登樓所見，抒發對時局的感慨。
- 出處：本詩選自《杜少陵集》。

**三、題解**

本詩記述作者登樓居高臨下眺望家國，引發聯想和感慨。

**四、文體**

- 近體詩（七言律詩）
- 句式：全詩八句，每句七字
- 對仗：頷聯（第三至四句）、頸聯（第五至六句）對偶

**五、課文語譯**

|  |  |
| --- | --- |
| 花近高樓傷客心，  萬方多難此登臨。  錦江春色來天地，  玉壘浮雲變古今。  北極朝廷終不改，  西山寇盜莫相侵。  可憐後主還祠廟，  日暮聊為《梁甫吟》。 | 繁花在高樓近處盛放，我在這裏作客，觸景傷情。  在國家多處地方危難重重的時刻，我到此處台登高望遠。  錦江一帶的春色，已鋪天蓋地湧來。  玉壘山的浮雲有如古往今來的世情不斷變幻。  我們的朝廷會像北極星一樣始終不會改變。  西山來的強盜，不得再來侵犯。  可歎像蜀國後主劉禪那樣的人，仍有祠廟來紀念他。  我在這黃昏時分，姑且唸唸《梁甫吟》吧。 |

註釋：

1. 北極：北極星，比喻北方的朝廷。

2. 西山寇盜：指吐蕃。

3. 後主：三國時期蜀漢的後主劉禪。

4. 《梁甫吟》：一作《梁父吟》。據《三國志‧蜀志》，諸葛亮在隆中躬 耕時，好為《梁父吟》。

**九、問答**

1. 首聯的「花近高樓」本是春天繁花盛放之景，為甚麼卻「傷客心」？

首聯杜甫寫自己登樓，繁花滿眼，一片生機，正如唐室收復長安，一片好景。但他隨即想到時局多變，國家多難，而自己仍作客他鄉，歸家無期，報國無路，所以說「傷客心」。

1. 試分析以下詩句中**粗體字**所藴含的意義和感情。

|  |
| --- |
| 甲　錦江春色**來**天地，玉壘浮雲**變**古今。 （頷聯）  乙　北極朝延**終**不改，西山寇盜**莫**相侵。（頸聯）  丙　可憐後主**還**祠廟，日暮**聊**為《梁甫吟》。（尾聯） |

1. 甲句中，「來」字富① 動感，寫錦江流水② 挾着蓬勃的春色從天地邊際洶湧而來，有春臨大地、③ 勢不可擋之感。「變」固然寫盡④ 玉壘山浮雲的變化，但「變古今」更有閱盡古往今來世事變幻的感覺，引發⑤ 人對歷史的無窮聯想。
2. 乙句中，「終」字帶有① 肯定的語氣，杜甫相信唐室的政權會② 長久地穩固下來。「莫」字帶有警告的語氣，流露杜甫對吐藩入侵唐室的③ 憤慨和嚴言斥責。這兩句詩都反映杜甫對④ 國家和人民的關愛。

（3） 丙句中，「還」」字含有「不當如此而居然如此」的語氣，流露①杜甫對誤國昏君的不滿。「聊」字含有② ● 不甘／ ○ 感歎 的語氣，表達③杜甫憂時不遇的無奈感慨。

**十、賞析重點**

- 此詩題為「登樓」，然重點並非描寫登樓所見，杜甫只是借登樓所見，抒發對時局的感慨。
- 開首兩句起勢極其高遠。花近高樓使人傷心，是因為春光再度，而詩人依然客居在外，在天下多難之際登臨此樓，心情自然不言而喻。首句雖是眼前之景，但次句即拓出遠勢，以「萬方多難」作為登臨的背景，將整個多災多難的時代都拉到了眼前。有此起勢，方能展開下句更加壯闊的境界：錦江的春色鋪天蓋地而來，玉壘山的浮雲自古至今不斷變化。這兩句以江水和山雲、天地和古今相對，囊括時空，筆力雄壯，不但以「俯視宏闊、氣籠宇宙」（王嗣奭《杜臆》）的氣象為後人稱頌，而且蘊含着深刻 的寓意：天地春來，與「花近高樓」照應，是亙古常新的江山；古今浮雲與「萬方多難」呼應，是變化不斷的時事。玉壘山在蜀中和吐蕃的交通要 道上，詩人刻劃山上浮雲飄游不定，是寫實景，也是象徵捉摸不定的時勢變化。
- 當時朝廷剛收復長安，吐蕃又新陷三州；自初唐以來，唐與吐蕃以及周邊民族的關係，一直處於反覆不定的狀況。不過，這並沒有動搖詩人春色常在的信心，因此下面再以「北極朝廷」和「西山寇盜」作人事的對比，指出儘管吐蕃不斷相侵，朝廷仍不會受到動搖，就如北極星永遠不會 移動，春色照常會降臨人間，不過當中又隱含對今後局勢的憂慮。以上六句，意脈相連，一、三、五句就朝廷春色而言，二、四、六句就寇盜侵擾而言，形成同一意思的三層對比，而這三層的對比綜合運用了興、比、賦三種藝術手法來表現，花近高樓是借物起興，登樓所見是景中寓比，聯繫時勢是賦陳其事。
- 尾聯借古跡以詠懷。詩人登樓所見，後主的祠廟就在武侯祠附近，由此想到劉禪亡國後被送到洛陽，樂不思蜀，但他的祠廟卻建在蜀中，因而 發出感歎，並由此想到蜀國賢相諸葛亮。如果聯繫現實來看，詩人是藉劉 禪信任宦官而亡國一事，暗諷代宗信任宦官程元振而招致禍亂。詩人也因而慨歎當世沒有如諸葛亮的人才輔國。《梁甫吟》是諸葛亮躬耕南陽時愛吟唱的歌謠，詩人日暮時吟起這首詩，一方面是懷念諸葛亮之雄才偉略， 另一方面也是寄望朝廷能任用賢才，振興國運。詩人同時期的作品《傷春》(其三)，同樣是感慨時局之作，可以與本詩互相參照，「不成誅執法，焉得變危機？……賢多隱屠釣，王肯載同歸？」也是批評代宗不肯除掉亂政的程元振，不起用隱潛在民間的賢人。
