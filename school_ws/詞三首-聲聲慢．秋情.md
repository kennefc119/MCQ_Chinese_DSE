1. **作者簡介**
2. 李清照（1084–約1155），自號易安居士，齊州 章丘（今山東 濟南）人，宋代女詞人。
3. 生平：李清照生於愛好文學的士大夫家庭，18歲與趙明誠結婚，夫婦志趣相投，詩詞唱和，收集金石書畫，生活美滿。宋欽宗靖康二年（1127），金兵攻陷汴京，宋室南渡，她與丈夫趙明誠南遷到江寧（今江蘇南京），兩年後趙明誠病死。李清照晚年居於江南越州（今浙江紹興）、杭州、金華等地，孤苦無依。
4. 風格：她的作品可按生平遭遇分前後兩期，前期多寫愛情相思，閨怨離愁；後期多 寫飄泊、懷鄉之苦，個人悲慘遭遇與國家興亡交織，有《漱玉詞》傳世，是婉約派的大家。

**二、寫作背景**

- 這首詞作於李清照晚年寓居臨安時創作，屬於後期作品。
- 北宋亡國後，宋室南渡，李清照隨丈夫趙明誠南下，宦居江寧、湖州。
- 自丈夫趙明誠去世後，李清照流居越州、杭州、金華等地，飽嘗顛沛流離的艱苦，境況極為淒涼。亡國的恨、喪夫的哀、孀居的苦凝集心頭，無法排遣，於是寫下了這首《聲聲慢．秋情》。

**三、題解**

本詞描寫秋日從早到晚一天的景況，抒發自己隨宋室南渡後國破家亡、流落天涯之痛苦。

**七、課文問答**

1. 在本詞，作者如何通過描寫秋天景況渲染秋意，以抒發感情？試加以說明。

詞的上片：作者表示秋天天氣忽冷忽熱，暗示作者心情起伏不定，滿懷愁思；描述寒風急勁，抒發內心淒涼之感；描述雁兒南飛，觸發作者因故都淪陷金人之手，國破家亡，抒發懷念故土和流落異鄉之悲苦。

詞的下片：描述庭院滿地凋零的黃花，抒發自己年華老去，因思念丈夫和懷念故土，以致容顏憔悴，孤苦無依的淒涼，描繪淅淅瀝瀝的雨聲打在梧桐葉上，猶如打在自己心間，綿綿不盡的雨聲在夜深人靜之時使人難以安眠，抒發作者孀居的孤寂和對亡夫的思念。

2. 在「梧桐更兼細雨，到黃昏、點點滴滴」（下片）一句中，李清照如何將「梧桐」和「細雨」所抒發的感情相扣連？

黃昏細雨連綿，營造淒冷氣氛；雨打梧桐，自古是孤寂、哀愁的象徵。李清照描繪淅淅瀝瀝的雨聲打在梧桐葉上，細寫雨滴梧桐之聲，猶如打在自己心間，「到黃昏」中「到」字刻畫雨滴之久，綿綿不盡的雨聲在夜深人靜之時使人難以安眠，更添愁思。此句情景交融，抒發作者孀居的孤寂和對亡夫的思念。

3. 以下句子分別為上片和下片的末句，它們在全詞的結構上各起甚麼作用？

|  |
| --- |
| 甲　雁過也，正傷心，卻是舊時相識。  乙　這次第，怎一箇愁字了得！ |

① 甲句：暗寫李清照從北方流徙南方的遭遇，總結上片詞中愁情的根源／起因，引入下片自傷身世的感慨，有承上啟下的作用。

② 乙句：總結上文所描寫的光景，慨歎自己孤獨無依和寂寞悲苦，不是一個愁字能說得盡的，全首詞雖不提及「愁」字，但所實內容處處隱含愁緒，末句達首尾呼應的作用，使愁思更為深刻。

4. 上片「晚來風急」一句，有人認為應改為「曉來風急」，也有人不同意。試分別指出雙方的理據。

|  |
| --- |
| 提示：本詞記述的是甚麼時間？當中描寫了甚麼景象？如果改成「曉來風急」後，對描寫的景象有怎樣的影響？是否較「晚來風急」更恰當？ |

同意的一方認為「乍煖還寒」應指秋日清晨，朝陽初出，天氣和暖，但曉寒猶重，而「淡酒」指早上喝的酒。全詞描寫作者從早晨到黃昏，一整天之中的感受，而非一晚之事。所以應改為「曉來風急」，這樣能較有層次地刻畫秋天由早到晚的景象。

而不同意的一方認為晚風與下文的「黃昏」相呼應， 可見全詞主要寫秋日傍晚之景。而且黃昏時分太陽下山，急風送寒，符合文中「乍煖還寒」的温差特點，所以「晚來風急」更恰當。

5. 若將「雁過也，正傷心，卻是舊時相識」改為「雁過也，更傷心，因是舊時相識」是否更理想？試結合李清照所抒發的感情，談談你的看法。

會更理想。因為李清照看見雁過景象之前，心情已十分憂愁傷感，所以當她看見雁過時應「更」傷心。由於見到舊時在北方相識的大雁，內心更添愁思，「舊時相識」與「傷心」為因果關係，「因」一字則更能帶出這個關係。

或

不會更理想。因為原句中「正」、「卻」二字表現了情感的層層遞進。李清照看見北雁南飛，正為自己流寓南方的身世而傷心，又發覺大雁是舊相識，再次觸動內心的愁緒，令悲愁更深。

6. 李清照的作品以善用口語著稱，試以本詞為例，說明這個風格特點。

李清照在本詞上片，以「怎敵他晚來風急」一句直接指出自己敵不過秋風，暗示受到愁緒所影響。在詞的下片，詞人以「守着窗兒，獨自中怎生得黑」直率地表示自己受不了孤身一人，感到度日如年，難以熬至黑夜； 末句「這次第，怎一箇愁字了得」則直接道出她的愁思太深，不能單以一個「愁」字概括得了。李清照在詞中通過不同的口語直抒情感，突出自己深深的愁思，顯得真情流露，讓讀者能切實感受到她的哀愁。

1. **賞析重點**

《聲聲慢》寫李清照晚年無可排遣的悲愁和孤寂，是她後期最受人稱道的作品。清代彭孫遹評為「皆用淺俗之語，發清新之思，詞意並工，閨情絕調」（《金粟詞話》）。

這首詞的上片起首三句連用七組疊字，歷來備受激賞。這七組疊字包含着豐富的情感層次：「尋尋覓覓」寫若有所失的空虛落寞之情，把她因北宋的覆亡、丈夫的身故，以及流離失所而不得不承受的一切悲痛，充分展示出來。緊接的「冷冷清清」寫尋覓無着的徹骨清冷，而難耐的孤冷寂寞在詞人心中反復迴旋強化，終於轉為「悽悽慘慘戚戚」。這三組疊字利用音調的三次轉折，直白地傳達出情感從失落轉為悽戚、再由悽戚轉為悲痛的過程。七組疊字運用了大量舌齒音和齊微韻，由外而內，由弱到強，由聲及心，沉痛無比，確實是「千古創格」、「絕世奇文」。滿懷愁苦的日子本來就難以打發，「乍煖還寒時候，最難將息。三杯兩盞淡酒，怎敵他，晚來風急！」人在孤苦寂寞之中，對天氣的變化總是特別敏感的，更何況在這深秋黃昏急風送寒之際。「乍煖還寒時候」如指季節，一般應在春天。但此詞明言秋季，所以有學者認為應指一天之內氣溫乍煖乍寒的變化，這也言之成理。淡酒不敵晚來風急，固然是難以禦寒之意，但真正不敵的還是內心的悽寒，這正是婉約詞的含蓄表達方式。正在傷心之際，卻看到一行征雁飛過長空，「卻是舊時相識」也很耐人尋味：首先，在李清照早年的詞中也常寫到大雁，如《一剪梅》「雲中誰寄錦書來？雁字回時，月滿西樓」。當年李清照在北方生活，大雁是給她帶來丈夫消息的信使；如今這群似曾相識的大雁又飛來了，但那曾經託牠們捎信的人已經不在了。再者，中原淪陷，李清照流落南方，無法重返故土，見到這些從北方南下的大雁難免再度勾起故土之思。這些大雁是詞人在「所遇無故物」的孤寂生活中所看到的唯一舊相識了，但是它們無知無識，一掠而過，徒然讓人更加傷心、惆悵。

下片繼續抒發愁情。「滿地黃花堆積，憔悴損」可理解為遍地菊花已凋謝枯萎，凋殘的花瓣堆積滿地，已經無法摘取來簪髮或者把玩了；但也有學者指出菊花枯萎時花瓣不落，因此「堆積」不是指凋落的花瓣，而是指菊花開得正盛。菊花品種很多，確有一些黃菊在盛開時非常繁密，像是堆在地上一樣，以「滿地堆積」一詞形容極為貼切。有人認為「如今有誰堪摘」是指現在沒有可以摘取黃花送與的人了，這是表達對丈夫的懷念。「憔悴損」三字既可指花由盛轉衰，也可以指人的憔悴。這時的李清照已經人到中年，甚至是老年，飽經離亂，身心疲憊，豈不正像那憔悴傷損的黃花？如果聯繫她前期所作《醉花陰》中的名句「莫道不銷魂，簾卷西風，人比黃花瘦」來看，她早就曾將自己與黃花進行比照，不過當年的「人比黃花瘦」只是閨中少婦甜蜜的相思致人消瘦，如今卻真的是人與黃花一樣憔悴了。「守着窗兒，獨自怎生得黑！」面對那令人傷心的秋景，詞人希望黑夜盡快來臨；但是，獨自守着寒窗，時間難熬。好不容易才到黃昏，卻又有令人煩惱的「梧桐更兼細雨，到黃昏、點點滴滴」。細雨打在梧桐葉上，在寂靜的夜晚，聲聲入耳，更添惆悵。這三句也是此詞膾炙人口的名句，除了再一次使用疊字之外，更重要的是「梧桐更兼細雨」這一意象的組合，具有很強的感染力。梧桐細雨不是李清照的原創，唐代詩詞中就屢見不鮮，最著名的如白居易《長恨歌》中的「春風桃李花開日，秋雨梧桐葉落時」，以及溫庭筠《更漏子》中的「梧桐樹，三更雨，不道離情正苦。一葉葉，一聲聲，空階滴到明」；到宋代則又有晏殊《踏莎行》中的「高樓目盡欲黃昏，梧桐葉上蕭蕭雨」，以及歐陽修《一落索》中的「窗在梧桐葉底，更黃昏雨細」等。李清照利用了同一個意象組合方式，改變其句式，置於具體的語境中，便產生了不同的強烈效果，令人想到這從黃昏就開始滴滴瀝瀝的雨聲，給詞人帶來一個無眠的長夜。而這聽雨的滋味，正如後來元代的白樸在《梧桐雨》雜劇中所說：是「驚魂破夢，助恨添愁，徹夜連宵」，人的心都被這雨聲一點點地滴碎了，也正因如此，李清照到最後才說：「這次第，怎一箇愁字了得！」這種種情景所帶給人的，怎可以用一個「愁」字概括得了！下片一層緊逼一層地訴說無法排解的悽寂之後，末句從難以抑制的一腔愁情中迸出的最後一聲哀歎。

清代評論家陳廷焯盛讚「後幅一片神行，愈唱愈妙」（《雲韶集》）。通觀全詞，句句寫愁，由愁及景。從頭至尾幾乎全是直白的訴說，所有景致都在愁人的怨艾和反問中見出，卻依然含蓄婉轉，曲折盡致；更兼語言淺易，又全押入聲險韻，使節奏較慢的長調，讀來十分急促，因而聲情並茂，極盡低迴抑塞之情態。李清照長於白描和直抒胸臆，這樣的技巧在這首詞裏達到了爐火純青的境界。

(iii) 作者看到鴻雁後勾起了什麼愁緒？為什麼？試加以說明。(4分)

評分原則：

能就鴻雁的意象或象徵， 準確說明兩項勾起的愁緒及其原因，闡述飽滿，每項2分。

參考答案：北雁南飛跟詞人國破家亡流落南方的境況相似，眼見舊時相識，想起雁

兒仍有北返歸家之日，而詞人卻因戰亂而歸家無期，心生身世飄零之嘆。鴻雁為信使，睹物思人，追憶昔日與丈夫相處的美好時光，今孤苦無依，徒增孀居之苦。

「鴻雁」的意象或象徵舉隅：

- 鴻雁是群居的冬候鳥，每年隨季節遷移，春來北國，秋去南方。故常引起遊子思鄉和羈旅之愁。古人常以鴻雁春秋遷徙，比喻人生漂泊無定，無可依靠，淒涼悲苦的遭遇。
- 古人分隔異地，以雁書寄託思親懷鄉之情，故常以鴻雁喻家書。又大雁群飛的時候，在天空排列成「人」字的形象，故又可用以隱喻思念的對象。

鴻雁勾起愁緒的原因舉隅：

- 故國之思：中原淪陷，詞人流落南方，無法重返故鄉。北雁南飛尚有回歸北方的一天，自己卻流離南方，回鄉渺茫，故勾起故國之思和身世飄零的感慨。
- 喪夫之痛：鴻雁本傳書信，可是與丈夫生死永隔，再也收不到書信。北雁南飛，在孤苦寂寞時見到舊時相識，令人追憶往日美好的時光，徒添傷心、惆悵。
