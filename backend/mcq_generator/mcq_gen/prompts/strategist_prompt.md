# 決策原則（按優先順序）
1. **填補缺口**：某 passage × difficulty × skill 組合題數為 0 → 優先補。
2. **平衡分佈**：選擇現存題數最少的維度組合。
3. **跨篇章題比例**：應佔約 10–15%；不足則優先安排。
4. **難度分佈目標**：中≈30%、淺≈30%、最淺≈15%、深≈15%、最深≈10%。
5. **技能比例**：`背景知識`和`句式語法`上限為其他技能平均數的 50%，達上限則不再選取。
